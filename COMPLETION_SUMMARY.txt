PAYMENT PROCESSING LOGIC
================================================================================

CASH:
  Amount = Item prices
  No charges

CARD:
  If Amount < ‚Çπ20,000:
    Amount = Item prices
    No charges
  
  If Amount ‚â• ‚Çπ20,000:
    Amount = Item prices √ó 1.03 (add 3% bank charge)
    
KOKO (Special):
  Only for Silver jewelry < ‚Çπ20,000
  System prevents use outside these parameters

FINAL BALANCE:
  Balance = Final Amount - Old Gold Value

================================================================================
WASTAGE CALCULATION FORMULAS
================================================================================

THEORETICAL WASTAGE:
  = Target Weight √ó Wastage Coefficient

ALLOWED WASTAGE:
  = Final Metal Weight √ó Wastage Coefficient

ACTUAL WASTAGE:
  = Gold Given - Final Metal Weight - Purity Corrected Balance

WASTAGE STATUS:
  Ideal:  Difference = 0
  Low:    Difference < 0
  Excess: Difference > 0

COEFFICIENTS BY METAL:
  Gold 22K:    0.0562
  Gold 21K:    0.06
  Gold 18K:    0.1
  Gold 16K:    0.12
  Gold 14K:    0.13
  Gold 9K:     0.15
  Silver 925:  0.1
